This document describes algorithms for the
spatial analysis package of the ROSAT/IRAF Post Reduction Off-line
Software (the PROS). The chapters of this document were
written by Diana Worrall, implemented by Mike VanHilst, Maureen
Conroy, Janet DePonte, and John Roll, and the document has become
a `working document' which is under frequent revision.  Please check
with Diana that you have the latest version, and inform her of
errors.
\blankline
Note that version numbers and dates heading even-numbered pages refer
to the last date on which the content of the corresponding
chapter was significantly modified.
